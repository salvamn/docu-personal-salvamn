HTML 5 : Estructura basica de un documento



HTML de las siglas en ingles HyperText Markup Language, html es el lenguaje para el desarrollo web, este nos permite darle una estructura a nuestra pagina. Como podemos observar en la imagen de abajo tenemos lo que es una estructura para un pagina web similar en la que estamos navegando en este momento(Esta estructura no es la definitiva ni la unica, cada desarrollador puede diseñar su propia estructura).


en el ejemplo que podemos observar abajo tenemos una estructura basica de html, la etiqueta DOCTYPE hace referencia a que se esta usando un archivo html 5, seguido viene la etiqueta HTML que declara que su contenido debe ser interpretado como codigo html, el HEAD es el lugar donde configuraremos algunos aspectos de nuestro documento y el BODY es el cuerpo donde agregaremos muchas otras etiquetas. 


                    
    <!DOCTYPE html>  
    <html lang="es">
    <head>
        <title>Titulo de la Pagina</title> 	
    </head>
        <body>
        </body>
    </html>

Etiquetas y sus usos


Texto

<p> : La etiqueta p representa un parrafo.

                            
<p>Hola soy un parrafo </p>

Resultado

Hola soy un parrafo


<pre> : La etiqueta pre nos permite mostrar codigo de programacion sin que este intervenga en el documento.

<code> : La etiqueta code representa codigo de programacion se usa junto a la etiqueta pre.

                            
<pre>
    <code>
        print("Hola mundo, soy codigo de python")
    </code>
</pre>

Resultado

print("Hola mundo, soy codigo de python")

<span> : La etiqueta span uno de sus usos es para aplicar estilos personalizados.

                            
<p>
    Soy el parrafo
    <span style="color: blue">
        Soy un span personalizable.
    </span>
</p>

Resultado

Soy el parrafo Soy un span personalizable.

<br> : La etiqueta br nos permite agregar un salto de linea.

<wbr> : La etiqueta wbr hace que el navegador opcionalmente agregue un salto de linea donde lo crea necesario.

<em> : em agrega un enfasis al texto, muesta el texto en cursiva.

<strong> : strong nos permite colocar un texto o una palabra en negrita.

<i> : La etiqueta i muestra el texto marcado con un estilo en cursiva o italica.

<u> : La etiqueta u muestra el texto o la palabra en subrayado.

<b> : Esta etiqueta b se usa para indicar importancia al texto o parrafo dajandola en negrita.

<mark> : Esta etiqueta mark nos permite resaltar un texto o frase relevante.

<small> : La etiqueta small cambia el tamaño de una letra a pequeña.

<cite> : La etiqueta cite se usa para representar una opinion, un autor, una obra, una pelicula, etc.

<address> : La etiqueta address se puede usar para representar una direccion, frecuentemente se usa en el footer.

<time> : Esta etiqueta time representa una fecha en formato legible para el usuario, 2021-05-08.

<data> : Esta etiqueta data representa datos genericos.



Enlaces

<a> : La etiqueta a o conocida tambien como anchor nos permite crear un enlace que al presionarlo puede abrir una pagina, una imgen o hasta descargar un archivo.



Imagenes

<img> : La etiqueta img nos permite mostrar una imagen en el documento.

<picture> : picture nos sirve para mostrar imagenes en multiples resoluciones, su uso comun es al momento en el que se necesita hacer una web responsiva.

<figure> : La etiqueta figure epresenta contenido independiente, a menudo con un título. Si bien se relaciona con el flujo principal, su posición es independiente de éste. Por lo general, se trata de una imagen, una ilustración, un diagrama, un fragmento de código, o un esquema al que se hace referencia en el texto principal, pero que se puede mover a otra página o a un apéndice sin que afecte al flujo principal.

<figcaption> : La etiqueta figcaption nnos permite agregarle un titulo auna imagen por ejemplo.



Listas

<ul> : La etiqueta ul crea una lista desordenada.

<li> : La etiqueta li nos permite definir un elemento de la lista, esta etiqueta va en el cuerpo de la etiqueta ul o ol.

<ol> : La etiqueta ol crea una lista ordenada.

<dl> : La etiqueta dl crea una lista de terminos y descripcciones.


Esto es una lista desordenada

  • Item 1
  • Item 2
  • Item 3

Esto es una lista ordenada

  1. Item 1
  2. Item 2
  3. Item 3


Tablas

<table> : Esta etiqueta table define una tabla.

<tr> : La etiqueta tr define una fila de celdas en una tabla.

<th> : La etiqueta th define una celda para la cabecera de nuestra tabla, y pinta en negrita el contenido dentro de ella.

<td> : La etiqueta td define una celda para nuestra tabla

Formularios

Los formularios permiten que los usuarios puedan navegar en nuestra web e introducir informacion que podemos enviar al servidor y procesarla.


<form> : La etiqueta form define un elemento de tipo formulario.

<input> : La etiqueta input crea un capo de tipo entrada o tambien conocido como caja de texto.

<textarea> : La etiqueta textarea crea un campo para multiples lineas.

<select> : select crea un combobox con opciones seleccionables.

<button> : La etiqueta button como su nombre lo dice crea un boton.
